Atividades

Método de Newton

O Método de Newton (Diferenças Divididas), é uma técnica iterativa para encontrar aproximações das raízes de uma função real f(x)=0.

Este método é conhecido por sua rápida convergência quando a estimativa inicial está próxima da raiz e a função é bem comportada.

Método de Lagrange

O Método de Lagrange é uma técnica para interpolação polinomial, usada para encontrar o polinômio de grau n que passa por n+1 pontos dados.

Este método é útil para aproximar funções e valores desconhecidos dentro do intervalo dos pontos conhecidos.

Folha da Árvore

Nessa atividade prática, utilizamos uma folha qualquer para aplicarmos na prática o Método de Newton.
Primeiramente tiramos algumas medidas (x, y) da folha utilizando a régua, após isso, escolhemos um valor entre os pontos medidos para que fosse interpolado; feita a interpoação, validamos com a medida real da folha, assim, verificando a margem de erro com base nos dados fornecidos e calculado.

Dados Medidos
Pontos escolhidos:
(1,7 ; 1,29)
(3,1 ; 1,70)
(6,0 ; 1,70)
(7,0 ; 1,20)
(8,9 ; 0,30)

PONTO PARA INTERPOLAR:
x = 5,20
y = 1,717049721715434

PONTO REAL:
x = 5,20
y = 1,75

ERRO:
~1,9% (aproximado)
